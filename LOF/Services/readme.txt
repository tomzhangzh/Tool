第一步：准备运行环境
安装 Chrome 浏览器
确保你的电脑已安装最新版 Chrome 浏览器（下载地址）。
下载 ChromeDriver 驱动
查看 Chrome 版本：打开 Chrome → 右上角三点 → 帮助 → 关于 Google Chrome
下载对应版本的 ChromeDriver：ChromeDriver 官网
解压后将chromedriver.exe文件放到桌面（方便后续使用）。
